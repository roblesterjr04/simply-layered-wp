== Description ==

Layered Bliss is the simple, elegent, modern, and fully responsive theme you've been looking for. Built out of necessity, its mobile friendly and very flexible. If you haven't noticed, this site is running on it. It works great with the PageBuilder plugin by Site Origin.

* Responsive
* Multipel Color Schemes
* Menu and Widget support
* 4 Widget Areas
* Works well with Site-Origin's Page Builder
* Custom Log In Screen Styles

== Installation ==

1. Navigate to Appearance > Themes
2. Click "Add New"
3. Click "Upload Theme"
4. Browse and select layered-bliss.zip
5. Activate
